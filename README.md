## Usage

## Install
- git clone https://github.com/hsynkkl/proje-doviz </br>


## Setup
- Install packages with `npm install`.


## Start
- cd proje-doviz/exchangeApp </br>
- npx react-native start </br>
- npx react-native run-android </br>
- cd proje-doviz/backend </br>
- nodemon index.js </br>
